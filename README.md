🛍️ ShopNest – Full Stack E-commerce Platform

ShopNest is a modern, full-stack e-commerce web application built with a premium black-and-gold UI theme. It delivers a complete online shopping experience with secure authentication, dynamic cart and wishlist management, multiple payment methods, and a powerful admin dashboard.

🚀 Features:
• User authentication with JWT
• Wishlist and shopping cart system
• Real-time cart & wishlist updates
• Secure checkout with multiple payment options
• Payment screenshot upload for manual verification
• Admin dashboard with order & user management
• Automated email notifications using NodeMailer

🛠 Tech Stack:
Frontend: HTML5, CSS3, JavaScript (ES6+)
Backend: Node.js, Express.js
Database: MongoDB Atlas with Mongoose

🔐 Security:
• Password hashing using Bcrypt
• Protected admin routes with JWT
• Environment variables for sensitive data

⚡ This project demonstrates full-stack development skills, API design, authentication, and real-world problem-solving.
