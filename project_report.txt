2. JAVASCRIPT FUNCTIONS (script.js)
--------------------------------------------------------------------------------

Core & UI Functions:
- initHeader() : Initializes scroll behavior for the site header.
- initSearch() : Handles search input focus, blur, and keyboard events.
- buildSuggestions(q, drop) : Generates live search results/suggestions.
- doSearch() : Executes the main search logic and redirects to the shop page.
- srchFor(term) : Shortcut helper to search for a specific term.
- nav(id, event) : SPA Router; handles page transitions and history.
- _showPage(id, scrollToTop, restoreScroll, savedPos) : Internal page render logic.
- navToSection(sectionId, event) : Smooth scrolls to home page sections.
- navToShopCat(cat, event) : Navigates to the shop and auto-filters by category.
- goBack(event) : Handles custom back navigation with scroll position restoration.
- toggleMNav() : Toggles the mobile navigation drawer.
- closeMNav() : Explicitly closes the mobile navigation drawer.
- toggleMobDrop(link, event) : Toggles category dropdown in mobile view.
- bounceCartBadge() : Animated feedback for the cart icon.
- showToast(msg) : Displays non-intrusive notifications.
- observeFadeEls() : Implements IntersectionObserver for entrance animations.
- updateActiveNavOnScroll() : Highlights the active navigation link during scroll.
- highlightMatch(text, query) : Highlights search keywords in results.

Product & Shop Functions:
- card(p) : Template literal function to generate a product card HTML.
- renderGrid(id, prods) : Renders a list of products into a specified grid.
- renderRelated(prods) : Renders the 'Related Products' section.
- filterTab(btn, cat) : Filters the home page featured products by tab.
- openDetail(id) : Populates and opens the product detail page.
- swGal(th, src, fb) : Switches the main image in the product gallery.
- pickSz(btn, label) : Handles size selection.
- pickCol(dot, name) : Handles color selection.
- chgQ(d) : Updates quantity counter for adding to cart.
- setView(btn, mode) : Toggles between Grid and List views.
- switchPg(btn) : Handles pagination page changes.

Filter & Logic Functions:
- applyAllFilters() : Main logic to combine Category, Brand, Rating, and Price filters.
- filterByCat(label) : Updates state for category filtering.
- filterByBrand(label) : Updates state for brand filtering.
- filterByRating(label) : Updates state for rating filtering.
- filterByPrice(val) : Updates state for price range filtering.
- applySort(val) : Re-renders the grid based on the selected sort order.
- clearF() / resetAllFilters(event) : Resets all filters to default state.
- togF(btn) : Toggles visibility of filter group bodies (accordion).

Cart & Review Functions:
- toggleCart() : Opens/Closes the cart sidebar.
- addToCart() : Adds current detailed product to cart.
- addByProd(id) : Adds a product to cart directly from grid.
- cQty(id, d) : Changes quantity of an item inside the cart.
- cRm(id) : Removes an item from the cart.
- renderCart() : Updates the cart UI with current state.
- togWl(btn) / togWish(btn, event) : Toggles wishlist status.
- openTab(id, btn) : Handles tab switching in product details (Description/Reviews).
- renderRevs() : Renders the customer reviews list.
- pickStar(n) : Handles star selection in review form.
- submitRev() : Processes and adds a new customer review.
- nlSub(event) : Handles newsletter subscription submission.
